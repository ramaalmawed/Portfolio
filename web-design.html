UX Thesis — H&M & Zara

My bachelor thesis explored how aesthetic and functional design principles affect the user experience on online fashion websites, using H&M and Zara as case studies.

Fashion E-commerce UX Research

This project focused on understanding how users experience fashion e-commerce websites. I studied how visual design, navigation, filters, product presentation, and usability influence the shopping experience.

The project included user testing, think-aloud sessions, observation, interviews, analysis, and UX evaluation of H&M and Zara.

RamaLyrics is a music lyrics application built with React. I designed and developed the interface with a focus on simple navigation, readable content, and a modern visual style.
