Alzenins Logo Design

A logo and visual identity system created for Alzenins, an online Japanese learning platform for Arabic-speaking students. The logo was designed to work across the website, learning materials, social media, books, notebooks, and printed products.

Project Context

Alzenins is a Japanese learning platform that I co-founded. I designed the logo in discussion with the other co-founders, while also considering the platform’s educational purpose, personality, and future use across different products.

My Role

I designed the logo system, explored different logo formats, chose the color direction, and prepared versions that could be used across digital and printed materials.

Collaboration

The final direction was developed through discussion with the co-founders to make sure the logo represented Alzenins as a learning platform and not only as a visual mark.

Purpose

The logo needed to feel recognizable, flexible, and professional while still keeping a creative and friendly feeling suitable for students.

Portfolio Note

I created more logo variations than shown here, but I only selected a few examples for the portfolio to keep the page clean and focused.

Logo Concept

Identity for a Japanese Learning Platform

The logo was created to represent Alzenins as a platform that combines learning, creativity, and Japanese language education. I wanted the identity to feel bold and memorable, but also friendly enough for an educational platform.

The strong letter “Z” became an important visual element because it can work alone as an icon, while the full “AIZEN” wordmark gives the brand a more complete and official identity.

I also designed the system so the logo can be used in different contexts: large website headers, small profile icons, book covers, teaching files, notebooks, and social media.

Logo System

Instead of creating only one logo, I designed a flexible logo system with several versions. This makes the identity easier to use across different formats and sizes.

Primary, Secondary, Stacked & Icon Versions

The primary logo works well for wide spaces, such as website headers and printed layouts. The stacked version is useful when the design needs a stronger centered composition, for example on covers or social media posts.

The icon version focuses only on the “Z.” mark. This version is useful for profile images, small labels, favicons, stickers, and places where the full logo would become too small to read.

I also created alternative color and black versions so the logo can still work on different backgrounds and in simpler print situations.

Logo Types: Primary Logo · Secondary Logo · Stacked Logo · Icon Logo · Black Version

Color Direction

Deep Blue

Used to communicate trust, stability, learning, and professionalism.

Soft Pink / Red

Used to add warmth, energy, creativity, and a more friendly student-centered feeling.

Black Version

Used for simplified applications, monochrome printing, or places where color is not suitable.

Balancing Professional and Friendly

The color palette was chosen to balance two sides of Alzenins: the serious learning side and the creative, friendly side. The deep blue gives the brand a stable and trustworthy feeling, while the pink/red tone makes it more warm and memorable.

I also created color variations where the main color changes depending on the use case. This gives the brand more flexibility without losing recognition.

Design Process

02

Exploration

03

04

Final System

I refined the final versions and prepared them for different uses such as website, books, teaching files, social media, and printed materials.
