Arab Kyoushi

A WordPress-based CMS concept created as a school project for Arab Kyoushi, the early version of what later became Alzenins. The project focused on creating a structured website for Japanese learning content, lessons, and educational resources.

Project Context

Arab Kyoushi was the earlier name and concept behind Alzenins. This WordPress project was created during my studies as a school project, so it was not used as the final live platform. However, it helped me explore how a learning platform could be structured, managed, and designed through a content management system.

Project Type

School project · WordPress website · CMS concept · Learning platform design

Brand Context

Created for Arab Kyoushi, which later developed into Alzenins.

Important Note

This version was not usable as the real platform because it was built as part of a university assignment, but the concept helped shape later ideas for Alzenins.

My Role

I worked on the structure, WordPress setup, page design, content organization, visual layout, and CMS thinking.

WordPress & CMS Work

Building with WordPress

The project was built using WordPress, with a focus on learning how content can be managed, organized, edited, and presented through a CMS. I worked with pages, menus, categories, content sections, and layout settings to create a website that could support Japanese learning content.

The goal was not only to design the visible website, but also to understand how a learning platform could be managed from the backend. This included thinking about how lessons, files, posts, and educational resources could be organized in a way that would be easy to update.

Focus: WordPress · CMS Structure · Page Design · Content Organization

Website Structure

The website was planned as a structured learning platform for Arabic-speaking students who wanted to study Japanese online.

Homepage

Introduced the platform, the learning idea, and the purpose of Arab Kyoushi.

Lessons

Planned sections for organizing Japanese lessons, levels, and learning materials.

Resources

Space for PDFs, learning files, grammar explanations, and student support material.

Navigation

Clear menu structure to make it easier for students to find content.

Admin Editing

CMS-based content editing so pages could be updated without writing code every time.

Visual Identity

A simple visual direction connected to Japanese learning and educational content.
